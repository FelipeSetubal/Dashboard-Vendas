# Dashboard de Vendas com Python

Projeto de dashboard interativo para análise de vendas, desenvolvido com Streamlit.

## Funcionalidades

* Upload de CSV
* Filtros por produto e data
* Métricas (faturamento, quantidade, ticket médio)
* Gráficos interativos
* Insights automáticos
* Download dos dados filtrados

## Tecnologias

* Python
* Streamlit
* Pandas
* Plotly

## Como rodar

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Exemplo de dados

O arquivo `data/exemplo.csv` pode ser usado para testes.

## Objetivo

Demonstrar habilidades em análise de dados e criação de dashboards interativos.
